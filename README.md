# SpeakPro 🎙

تطبيق PWA لتدريب النطق وتقوية الحفظ — يعمل بدون إنترنت.

## 📁 الملفات
```
speakpro/
├── index.html      ← التطبيق الرئيسي
├── manifest.json   ← بيانات PWA
├── sw.js           ← Service Worker (للعمل offline)
├── icon-192.svg    ← أيقونة صغيرة
├── icon-512.svg    ← أيقونة كبيرة
└── README.md       ← هذا الملف
```

## 🚀 كيفية النشر على GitHub Pages

1. افتح GitHub وأنشئ مستودعاً جديداً (مثلاً: `speakpro`)
2. ارفع جميع الملفات الخمسة داخل المستودع
3. اذهب إلى: **Settings → Pages → Branch: main → Save**
4. انتظر دقيقة ثم ستحصل على رابط مثل:
   `https://اسمك.github.io/speakpro/`

## 📲 تثبيته كتطبيق على الهاتف

### Android (Chrome):
1. افتح الرابط في Chrome
2. ستظهر رسالة "إضافة إلى الشاشة الرئيسية" في الأسفل
3. اضغط عليها → يُثبَّت كتطبيق

### iOS (Safari):
1. افتح الرابط في Safari
2. اضغط زر المشاركة ↑
3. اختر "إضافة إلى الشاشة الرئيسية"

## ✨ المميزات
- 🎤 تسجيل صوتي مستمر (لا ينقطع حتى تضغط إيقاف)
- 📖 وضع التدريب العادي مع الترجمة العربية
- 🔍 وضع الكشف (كلمة بكلمة لاختبار الحفظ)
- 🔊 زر استمع للنطق الصحيح (TTS)
- 📝 إضافة نصوصك الخاصة
- 📊 إحصائيات تفصيلية وكلمات تحتاج تدريباً
- 🌙 / ☀️ الوضع الداكن والفاتح
- 📴 يعمل بدون إنترنت بعد أول فتح
